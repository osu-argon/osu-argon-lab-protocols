# Small Jaw Crusher Method

## ID

`crush_jaw_small`

## Requires

`workspace_prep`
`jaw_small_clean`
`jaw_small_setup`
`jaw_small_crush`
`sieve_progression`
`sieve_methods`

## Description

This method uses a small jaw crusher with steel plates to progression decrease the grain size of a sample while progressively sieving.
